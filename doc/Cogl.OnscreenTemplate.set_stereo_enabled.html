set_stereo_enabled

Description:

public void set_stereo_enabled (Bool enabled)

Sets whether future Onscreen framebuffers derived from this template are attempted to be created with both left and right buffers, for use with stereo display.

If the display system does not support stereo, then creation of the framebuffer will fail.

Parameters:

this

A OnscreenTemplate template framebuffer
enabled

Whether framebuffers are created with stereo buffers
