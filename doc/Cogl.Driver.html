public enum Driver

Identifiers for underlying hardware drivers that may be used by Cogl for rendering.


Namespace: Cogl
Package: doc

Content:

Enum values:

  • ANY - Implies no preference for which driver is used
  • NOP - A No-Op driver.
  • GL - An OpenGL driver.
  • GL3 - An OpenGL driver using the core GL 3.1 profile
  • GLES1 - An OpenGL ES 1.1 driver.
  • GLES2 - An OpenGL ES 2.0 driver.
  • WEBGL - A WebGL driver.
