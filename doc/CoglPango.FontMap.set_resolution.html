set_resolution

Description:

public static void set_resolution (FontMap font_map, double dpi)

Sets the resolution for the font_map.

This is a scale factor between points specified in a FontDescription and Cogl units. The default value is 96, meaning that a 10 point font will be 13 units high. (10 * 96. / 72. = 13.3).

Parameters:

font_map

a FontMap
dpi

The resolution in "dots per inch". (Physical inches aren't actually involved; the terminology is conventional.)
