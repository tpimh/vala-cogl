Description:

public void foreach_output (OutputCallback callback)

Iterates all known display outputs for the given this and passes a corresponding Output pointer to the given callback for each one, along with the given user_data.

Parameters:

this

A connected Renderer
callback

A OutputCallback to be called for each display output
user_data

A user pointer to be passed to callback
