NO_AUTO_MIPMAP

Description:

NO_AUTO_MIPMAP

Disables the automatic generation of the mipmap pyramid from the base level image whenever it is updated.

The mipmaps are only generated when the texture is rendered with a mipmap filter so it should be free to leave out this flag when using other filtering modes
