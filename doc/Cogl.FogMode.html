FogMode

Description:

public enum FogMode

The fog mode determines the equation used to calculate the fogging blend factor while fogging is enabled.

The simplest cogl_fog_mode_linear mode determines f as:

  f = end - eye_distance / end - start

Where eye_distance is the distance of the current fragment in eye coordinates from the origin.


Namespace: Cogl
Package: doc

Content:

Enum values:

  • LINEAR - Calculates the fog blend factor as: 
      f = end - eye_distance / end - start
  • EXPONENTIAL - Calculates the fog blend factor as: 
      f = e ^ -(density * eye_distance)
  • EXPONENTIAL_SQUARED - Calculates the fog blend factor as: 
      f = e ^ -(density * eye_distance)^2
