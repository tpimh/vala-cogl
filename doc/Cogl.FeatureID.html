FeatureID

Description:

public enum FeatureID

All the capabilities that can vary between different GPUs supported by Cogl.

Applications that depend on any of these features should explicitly check for them using has_feature or cogl_has_features.


Namespace: Cogl
Package: doc

Content:

Enum values:

  • TEXTURE_NPOT_BASIC - The hardware supports non power of two textures, but you also need to check the cogl_feature_id_texture_npot_mipmap and cogl_feature_id_texture_npot_repeat features to know if the hardware supports npot texture mipmaps or repeat modes other than cogl_pipeline_wrap_mode_clamp_to_edge respectively.
  • TEXTURE_NPOT_MIPMAP - Mipmapping is supported in conjuntion with non power of two textures.
  • TEXTURE_NPOT_REPEAT - Repeat modes other than cogl_pipeline_wrap_mode_clamp_to_edge are supported by the hardware.
  • TEXTURE_NPOT - Non power of two textures are supported by the hardware.
  • TEXTURE_RECTANGLE - Support for rectangular textures with non-normalized texture coordinates.
  • TEXTURE_3D - 3D texture support
  • GLSL - GLSL support
  • ARBFP - ARBFP support
  • OFFSCREEN - Offscreen rendering support
  • OFFSCREEN_MULTISAMPLE - Multisample support for offscreen framebuffers
  • ONSCREEN_MULTIPLE - Multiple onscreen framebuffers supported.
  • UNSIGNED_INT_INDICES - Set if cogl_indices_type_unsigned_int is supported in Indices.
  • DEPTH_RANGE - cogl_pipeline_set_depth_range support
  • POINT_SPRITE - Whether set_layer_point_sprite_coords_enabled is supported.
  • MAP_BUFFER_FOR_READ - Whether map is supported with CoglBufferAccess including read support.
  • MAP_BUFFER_FOR_WRITE - Whether map is supported with CoglBufferAccess including write support.
  • MIRRORED_REPEAT - Whether cogl_pipeline_wrap_mode_mirrored_repeat is supported.
  • SWAP_BUFFERS_EVENT - Available if the window system supports reporting an event for swap buffer completions.
  • GLES2_CONTEXT - Whether creating new GLES2 contexts is suported.
  • DEPTH_TEXTURE - Whether Framebuffer support rendering the depth buffer to a texture.
  • PRESENTATION_TIME - Whether frame presentation time stamps will be recorded in FrameInfo objects.
  • FENCE -
  • PER_VERTEX_POINT_SIZE - Whether cogl_point_size_in can be used as an attribute to set a per-vertex point size.
  • TEXTURE_RG - Support for cogl_texture_components_rg as the internal components of a texture.
  • BUFFER_AGE - Available if the age of Onscreen back buffers are tracked and so get_buffer_age can be expected to return age values other than 0.
