PollFDEvent

Description:

public enum PollFDEvent

A bitmask of events that Cogl may need to wake on for a file descriptor.

Note that these all have the same values as the corresponding defines for the poll function call on Unix so they may be directly passed to poll.


Namespace: Cogl
Package: doc

Content:

Enum values:

  • IN - there is data to read
  • PRI - data can be written (without blocking)
  • OUT - there is urgent data to read.
  • ERR - error condition
  • HUP - hung up (the connection has been broken, usually for pipes and sockets).
  • NVAL - invalid request.
